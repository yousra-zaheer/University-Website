# University-Website

website live at: https://yousra-zaheer.github.io/University-Website/

A static university website with navigation options, built using HTML, CSS and JavaScript. The website has clickable buttons and nav-options that route to linked webpages.

## Features
- **Form Submissions**: Contact and comment forms work with Formspree.
- Responsive navigation and buttons.

## Tech
- HTML, CSS, JavaScript
- Formspree for form handling
